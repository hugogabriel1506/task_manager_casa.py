# Gerenciador de Tarefas em Python

## O que o meu programa faz
- Permite adicionar tarefas
- Mostra todas as tarefas cadastradas
- Permite remover tarefas
- Tem um menu interativo para o usuário escolher as opções

## Como usar
1. Abrir o terminal na pasta do projeto  
2. Executar o arquivo Python:
3. Escolher a opção desejada no menu

## Testes que eu fiz
Durante o desenvolvimento, testei algumas situações:
- Adicionar tarefa normalmente
- Tentar adicionar tarefa vazia
- Listar tarefas quando não tem nenhuma
- Remover tarefas existentes
- Digitar opções inválidas

## Dificuldades
No início tive um pouco de dificuldade com:
- Trabalhar com listas
- Validar o que o usuário digita
- Organizar o menu com `while`
